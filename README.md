지난 4주간 배운 JAVA, DB를 바탕으로 실제 백엔드 애플리케이션을 직접 만들어본다.

### 🧑‍🤝‍🧑 맴버구성

 - 차현아 : 회원 API

### ⚙️ 개발 환경, 기술
- **JAVA 11**
- **IDE** : IntelliJ
- **Framework** : Springboot 2.7.14
- **Database** : MySQL
- **ORM** : JPA
- **DOCS** : Swagger
- **LOGGING** : Slf4J

- **FRONTEND** : REACT
